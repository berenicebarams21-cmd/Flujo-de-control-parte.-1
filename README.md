# Flujo-de-control-parte.-1
Calculadora de ISR en C++ 
Descripción.
Este proyecto es un programa en C++ que calcula el Impuesto Sobre la Renta (ISR) de un empleado a partir de su sueldo mensual.
Función.
Solicita el sueldo mensual. 
Calcula el sueldo anual.
Determina si aplica ISR.
Muestra el monto del ISR correspondiente.

C++
